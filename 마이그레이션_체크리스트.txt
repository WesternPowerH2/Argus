 PHASE 5: GitHub 및 자동 배포 설정
════════════════════════════════════════════════════════════════════════════════

□ 5.1 GitHub 저장소 초기화
   □ PowerShell (관리자) 열기
   □ cd C:\Argus
   □ git init
   □ git config user.name "이름"
   □ git config user.email "이메일"
   □ git remote add origin https://github.com/[계정]/argus.git
   □ git add .
   □ git commit -m "Initial commit"
   □ git branch -M main
   □ git push -u origin main

□ 5.2 PowerShell 스크립트 생성
   □ 파일: C:\Argus\깃헙_자동업로드.ps1
   □ CLAUDE.md Phase 5.2의 코드 참고
   □ YOUR_GITHUB_PAT 에 실제 토큰 입력

□ 5.3 Windows 작업 스케줄러 설정
   □ 작업 스케줄러 열기
   □ 작업 만들기:
      - 이름: Argus_GitHubPush_Daily
      - 트리거: 매일 09:20
      - 프로그램: powershell.exe
      - 인수: -ExecutionPolicy Bypass -File "C:\Argus\깃헙_자동업로드.ps1"

□ 5.4 GitHub Pages 활성화
   □ GitHub 저장소 > Settings > Pages
   □ Source: main branch, / (root)
   □ 저장 > 1~3분 후 포털 활성화
