# Argus 브리핑 포털 — 마이그레이션 필수 파일

## 📦 현재 PC에서 새 PC로 이전할 파일

### ✅ 반드시 복사할 파일 (핵심)

#### 프론트엔드 & 아카이브 데이터
```
현재 경로: D:\OneDrive\문서\claude\Argus\Argus 수소정보지 정리\
새 경로: C:\Argus\ (또는 선호 경로)

1. dashboard.html
   - 크기: ~150 KB
   - 설명: 아카이브 목록 및 가격 추이 차트의 핵심 DB
   - 역할: const briefs = [...] 배열에 모든 브리핑 메타데이터 저장
   - ⚠️ 주의: CSS/JS 구조 절대 변경 금지

2. index.html
   - 크기: ~2 KB
   - 설명: 포털 홈페이지
   - 역할: dashboard.html로 리다이렉트

3. login.html
   - 크기: ~3 KB
   - 설명: 로그인 페이지
   - 역할: 인증 후 dashboard.html로 이동

4. auth.js
   - 크기: ~8 KB
   - 설명: 클라이언트 사이드 인증 로직
   - 역할: 로그인 계정 검증 + 세션 관리
```

#### 설정 파일
```
5. .gitignore
   - 설명: Git 무시 목록
   - 내용: .gh_token, .DS_Store, Thumbs.db 등

6. .nojekyll
   - 설명: GitHub Pages Jekyll 비활성화 플래그
   - 크기: 0 바이트 (빈 파일)
```

#### 브리핑 아카이브
```
7. briefs/ 폴더 (모든 YYYYMMDD.html)
   - 크기: ~2~3 MB (모두)
   - 설명: 각 날짜별 브리핑 HTML
   - 예) 20260407.html, 20260408.html, ...
   - 파일 개수: 약 50~100개
   
   구조:
   briefs/
   ├── 20260407.html (암모니아 단독)
   ├── 20260408.html (암모니아 + 수소 탭 분리형)
   ├── 20260409.html
   ├── ...
   └── 20260619.html (최신)
```

#### Git 이력 (선택)
```
8. .git/ 폴더
   - 크기: ~1~2 MB
   - 설명: GitHub 커밋 이력
   - 역할: 향후 git log 조회용
   - 선택 여부: 이력 유지 원하면 복사, 아니면 스킵
```
